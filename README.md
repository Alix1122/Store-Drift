## 📖 À propos

DRIFT est une plateforme e-commerce complète dédiée aux vêtements et accessoires outdoor (goggles, vestes, pantalons, gants, casquettes, shorts).

Ce projet a été réalisé dans le cadre de mon **Projet de Fin d’Études (PFE)** en développement informatique.

---

## ✨ Fonctionnalités

### 👤 Côté utilisateur
- Catalogue produits par catégories
- Gestion des tailles : **XS, S, M, L, XL, XXL**
- Panier dynamique (ajout, modification, suppression)
- Validation de commande
- Inscription / Connexion
- Lightbox pour les images
- Animations au scroll
- Design responsive (CSS pur)

### 👑 Côté administrateur
- Dashboard avec statistiques (produits, commandes, CA)
- Gestion des produits (CRUD + upload image)
- Gestion des catégories (CRUD)
- Gestion des commandes
